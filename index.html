How to Start Using Git and GitHub?

Step 1 – Install Git

(https://git-scm.com/download)

and check, command line -> c:\>git version

Step 2 – Create a GitHub Account.

https://github.com

Step 3 – Connect GitHub account to Git account.

through terminal:

3.1 set your Git username: git config --global user.name "IbrahimElaraby"

check Git username: git config --global user.name

3.2 To set your Git email: git config --global user.email "mail@mail.com"

check Git email is set correctly: git config --global user.email

You will be asked to authenticate your GitHub account, so just sign in with the same email to confirm.
